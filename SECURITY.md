# Security Policy

## Supported Versions

Dieses Repository wird aktiv gepflegt. Sicherheitsprobleme sollten umgehend gemeldet werden.

## Reporting a Vulnerability

Wenn du eine Sicherheitslücke entdeckst, melde sie bitte so schnell wie möglich.

Bitte gib dabei folgende Informationen an:

- betroffene Datei oder Funktion
- Beschreibung der Schwachstelle
- Schritte zur Reproduktion
- mögliche Auswirkungen

Sicherheitsprobleme können über ein GitHub Issue im Repository gemeldet werden.

Gemeldete Schwachstellen werden geprüft und anschliessend so schnell wie möglich behoben.

Vielen Dank für deine Unterstützung bei der Verbesserung der Sicherheit dieses Projekts.
